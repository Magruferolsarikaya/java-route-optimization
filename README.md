# Java Route Optimization & Traffic Simulation

A console-based route optimization simulation built with Java. This project demonstrates the practical application of Graph Theory, specifically using **Dijkstra's Algorithm** to find the shortest path between nodes (cities) while dynamically calculating weighted edges (traffic penalties).

## 🚀 Features
* **Custom Graph Implementation:** Nodes and edges modeled using strict Object-Oriented Principles.
* **Dijkstra's Algorithm:** Utilizes a `PriorityQueue` to efficiently calculate the shortest path across the network.
* **Dynamic Traffic Penalties:** Simulates real-world conditions by allowing specific routes to have added "traffic weight", forcing the algorithm to dynamically recalculate and find alternative, optimal paths.
* **Path Tracing:** Reconstructs and displays the step-by-step route from the start node to the destination.

## 🛠️ Technologies & Data Structures Used
* **Language:** Java (JDK 8+)
* **Collections:** `HashMap`, `HashSet`, `PriorityQueue`, `ArrayList`
* **Concepts:** Weighted Graphs, Shortest Path Algorithms, Comparable Interfaces.
